# 打包
> 找到顶部工具栏Build菜单，选择 Generate Signed Bundle / APK

![链接](./img/3.png)
> 选择APK类型，点击Next

![链接](./img/4.png)
> 点击Next

![链接](./img/5.png)
> 选择 release 点击Create

![链接](./img/6.png)
> 等到页面底部的Build日志窗口提示 BUILD SUCCESSFUL in 387ms 点击右下角消息窗口的locate链接跳转打包目录

![链接](./img/7.png)

> 复制，回到本机，在需要的地方右键粘贴

![链接](./img/8.png)
